# A-mem

Localized fork of the A-mem agentic memory library (Zettelkasten-style semantic memory evolution), used as a dependency of the EvoMemBench benchmarks.

## Installation

Clone this repository, then install it in editable mode from the repo root:

```bash
git clone git@github.com:IcohM/A-mem.git
cd A-mem
pip install -e .
```

After installation, `from agentic_memory.memory_system import AgenticMemorySystem` should work in your Python environment.
